# Habilidad SOP Writer

## Cuándo activar
- Necesita documentar un proceso para que un nuevo empleado pueda seguirlo
- Redactar un manual del empleado o manual de operaciones
- Crear un script de servicio al cliente o una guía de respuesta de preguntas frecuentes
- Documentar su rutina de apertura/cierre, lista de verificación de limpieza o lista de verificación de calidad
- Preparar su negocio para funcionar sin usted

## Cuándo NO usar
- Procedimientos legales o requeridos por cumplimiento — debe ser revisado por un abogado
- Protocolos médicos o de seguridad — requieren revisión profesional
- Acuerdos de franquicia — use las plantillas de su franquiciador

## Instrucciones

### SOP básico de descripción

```
Redacte un SOP para: [describa el proceso en inglés claro]

Tipo de negocio: [su negocio]
Quién sigue esto: [rol/posición]
Frecuencia: [diariamente/semanalmente/cuando X ocurra]
Herramientas/sistemas utilizados: [lista]
Notas importantes o cosas que salen mal a menudo: [opcional]
```

### Procedimiento de apertura/cierre

```
Redacte un procedimiento de apertura para mi [restaurante/tienda/salón/oficina].
Hora de apertura: [hora]
Pasos que actualmente hago (en orden aproximado): [describe tu rutina]
Personal involucrado: [roles]
Formato: lista de verificación numerada con tiempo estimado para cada paso
```

### Script de servicio al cliente

```
Redacte un script de servicio al cliente para mi [tipo de negocio] que cubra:
1. Saludar a un nuevo cliente
2. Manejar la pregunta más común: [cuál es]
3. Tratar con una queja sobre [problema común]
4. Venta ascendente/recomendación de [producto/servicio]
5. Cerrar la interacción

Mi tono: [amistoso/profesional/entusiasta]
Los valores de mi negocio: [1-2 cosas clave]
```

### Lista de verificación de incorporación del nuevo empleado

```
Cree una lista de verificación de incorporación de la primera semana para un nuevo [rol] en mi [tipo de negocio].

Lo que necesitan saber:
- Nuestras herramientas principales: [lista]
- Sus responsabilidades principales: [describa]
- A quién reportan: [rol]
- La cosa más importante para hacerlo bien en la semana 1: [describa]

Formato: lista de verificación día por día para los días 1-5
```

### Documentación de proceso desde la entrevista

```
Voy a describir cómo hago [proceso] y quiero que lo conviertas en un SOP claro y paso a paso
que alguien más pudiera seguir.

Así es como lo hago: [describe de tu propia manera, tan desordenado como quieras]
```

### Lista de verificación de control de calidad

```
Cree una lista de verificación de control de calidad para [producto/servicio] antes de que se entregue a un cliente.

Qué se ve "bien": [describe tus estándares]
Problemas comunes que capturamos: [describa]
Quién verifica: [rol]
Formato: lista de verificación con criterios de aprobación/reprobación
```

## Ejemplo

**Usted dice:**
```
Redacte un SOP para manejar una queja del cliente en mi salón de belleza.
Quejas comunes: tiempo de espera, color no coincide con expectativas, sorpresa de precio.
Quiero que el personal se mantenga tranquilo, ofrezca soluciones, nunca discuta.
Podemos ofrecer: rehacer el servicio, descuento en la próxima visita, pequeño reembolso para problemas menores.
Tono: empático, profesional.
```

**Claude produce:**

**SOP: Manejo de Quejas de Clientes — Salón de Belleza**

**Propósito:** Resolver quejas profesionalmente, retener clientes, proteger la reputación del salón.

**Paso 1 — Escucha sin interrumpir**
- Haga contacto visual, asiente, deje que terminen completamente
- No defienda o explique aún
- Diga: "Siento mucho escuchar eso — gracias por decirme"

**Paso 2 — Reconozca el sentimiento**
- "Entiendo completamente por qué eso es frustrante"
- No diga: "Esa no es nuestra política" / "Pero te lo dijimos..." / "Eso nunca sucede"

**Paso 3 — Identifique el problema**
- Pregunte: "¿Puede ayudarme a entender exactamente qué sucedió?"
- Aclare: ¿es sobre el resultado, el precio, el tiempo de espera o la experiencia?

**Paso 4 — Ofrezca una resolución**
- Problema de color/resultado: ofrezca ajuste gratuito dentro de 7 días
- Sorpresa de precio: revisar conversación de fijación de precios; si es nuestro error, cumpla el precio más bajo
- Tiempo de espera: disculpa sincera + pequeño descuento en el servicio de hoy

**Paso 5 — Confirme la resolución**
- "¿Cree que es una solución justa?"
- Documente: nombre del cliente, fecha, tipo de queja, resolución en el sistema de reserva

**Paso 6 — Seguimiento (opcional pero poderoso)**
- Para quejas graves: el gerente llama o envía un mensaje dentro de 24 horas

**Nunca:** discuta, involucre a otros clientes, ofrezca reembolsos sin aprobación del gerente (sobre $50)
