# Ruby on Rails 7+

## Cuándo activar
- Construcción o mantenimiento de una aplicación Rails 7+
- Implementación de modelos ActiveRecord, asociaciones u optimización de consultas
- Adición de Hotwire (Turbo + Stimulus) para UI reactiva sin JavaScript pesado
- Configuración de trabajos de fondo Sidekiq
- Configuración de Devise para autenticación
- Escritura de especificaciones de solicitud RSpec con FactoryBot
- Despliegue con Kamal

## Cuándo NO usar
- Scripting Ruby puro sin participación de Rails
- Compilaciones solo de API donde Turbo/Stimulus se excluyen explícitamente
- Codebases Rails 6 o anteriores con convenciones significativamente diferentes
