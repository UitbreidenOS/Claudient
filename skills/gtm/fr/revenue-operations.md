# Compétence Opérations de Revenus

## Quand l'activer
- Construire un cadre de métriques RevOps (CAC, LTV, NRR, couverture de pipeline)
- Diagnostiquer pourquoi les prévisions de vente sont inexactes
- Concevoir le processus de vente et les définitions des étapes
- Analyser la santé du pipeline et identifier les goulots d'étranglement de conversion
- Mettre en place ou auditer un modèle de données CRM
- Construire un tableau de bord de métriques GTM pour la direction

## Quand ne pas l'utiliser
- Stratégie de transactions individuelles — c'est des ventes, pas RevOps
- Exécution de campagnes marketing — utiliser les compétences email-automation ou paid-ads
- Carnets de joueurs de réussite client — utiliser la compétence customer-success
- Décisions de tarification produit — utiliser la compétence pricing-strategy

## Instructions

### Cadre de métriques GTM

```
Construisez un cadre de métriques GTM pour [entreprise].

Type d'entreprise : [SaaS B2B / marketplace / services]
Motion de vente : [PLG / inside sales / field sales / dirigé par partenaires]
Stade : [pré-revenus / $0-1M ARR / $1-10M ARR / $10M+ ARR]
Équipe GTM : [fondateurs vendant / SDR + AE / équipe GTM complète]

Métriques GTM principaux par étape de l'entonnoir :

ACQUISITION :
- MQLs générés : [leads qualifiés marketing par mois]
- Taux de conversion MQL→SQL : [%] (référence : 10-25% selon l'ajustement ICP)
- CAC (Coût d'acquisition client) : [total des dépenses ventes + marketing / nouveaux clients]
- CAC par canal : [payant / organique / partenaire / outbound] — identifier le canal le plus efficace
- Temps avant premier contact : [heures du MQL à la sensibilisation SDR]

CONVERSION :
- Conversion SQL→Opportunité : [%]
- Taux Opportunité→Close-Won : [%] (référence : 15-25% pour inside sales, 10-20% entreprise)
- Taille d'affaire moyenne : [ACV]
- Longueur du cycle de vente : [jours d'opportunité créée à fermeture]
- Taux de victoire par segment : [PME / mid-market / entreprise] — où gagnez-vous le plus ?

RÉTENTION ET EXPANSION :
- NRR (Net Revenue Retention) : [%] (référence : >100% est sain, >120% est excellent)
- GRR (Gross Revenue Retention) : [%] — rétention pure sans expansion
- ARR d'expansion : [nouvel ARR de clients existants par mois]
- Taux d'attrition : [%] mensuel ou annuel
- Attrition de logo vs. attrition de revenus : [histoires différentes]

EFFICACITÉ :
- Ratio LTV:CAC : [cible >3x, >5x est fort]
- Période de récupération CAC : [mois pour récupérer le coût d'acquisition]
- Nombre magique : [nouvel ARR / dépenses ventes + marketing] (>0.75 est bon)
- Efficacité des ventes : [nouvel ARR / effectif porteur de quota]

Construisez le cadre de métriques avec les indices de référence actuels pour mon type et stade d'entreprise.
```
