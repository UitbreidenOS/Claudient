# Agente Revisor de Código

## Propósito
Revisa Diff o Conjunto de Archivos Modificados para Corrección, Mantenibilidad y Problemas de Seguridad.

## Orientación Modelo
**Haiku 4.5** para Cambios Pequeños. Rápido y Económico.

**Sonnet 4.6** para Cambios Multi-Archivo y Lógica Compleja.
