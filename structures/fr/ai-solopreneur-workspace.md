# 📂 AI Content Creator Studio
> L'espace de travail canonique pour un fondateur technologique en phase de démarrage ou un solopreneur IA, conçu pour orchestrer des agents de codage autonomes, lancer des produits physiques et numériques, et accélérer la monétisation.

📄 `founder-charter.md`        # Briefs canonique : objectifs commerciaux fondamentaux, définitions de pile technologique allégée et objectifs de revenus récurrents mensuels (MRR)
🧠 `active-sprint-memory.md`   # Mémoire de session : suivi de contexte dynamique pour les constructions autonomes de nuit et les campagnes de contenu actives
🤖 `CLAUDE.md`                 # Règles d'exploitation : instructions strictes pour optimiser la vélocité d'expédition et les opérations allégées plutôt que les frais généraux d'entreprise

## 📁 product-vision-and-monetization/ (4 compétences - Stratégie)
📄 `multi-agent-saas-blueprint.md` # Plans architecturaux pour une plateforme où les employés sont des agents IA, structurés spécifiquement pour monétiser rapidement
📄 `pricing-tier-calculator.md`    # Mappe les coûts de jetons aux niveaux d'abonnement pour assurer des marges brutes saines sur l'inférence IA
📄 `lean-canvas-generator.md`      # Extraction du modèle commercial sur 1 page à partir de notes de fondateur non structurées
📄 `competitor-gap-analyzer.md`    # Suive les solutions alternatives pour trouver les niches sous-desservies sur le marché des outils IA

## 📁 autonomous-agent-ops/ (3 compétences - La main-d'œuvre)
📄 `zaltaclaw-runbook.md`          # Moteur d'exécution de nuit permettant à l'agent IA autonome ZaltaClaw d'écrire, tester et refactoriser le code sans intervention humaine
📄 `claude-managed-agents.md`      # Spécifications d'intégration pour le suivi et le déploiement d'agents gérés via les API publiques bêta de Claude
📄 `mac-mini-host-config.md`       # Scripts pour allouer les charges de travail d'agents autonomes lourds à un Mac mini dédié pour économiser les coûts de calcul cloud

## 📁 hardware-and-prototyping/ (3 compétences - Produits physiques)
📄 `lifestyle-3d-designs.md`       # Plans CAO pour des articles de fantaisie fonctionnels (par exemple, porte-bouteilles de vin avec des personnages de culture pop comme Rick)
📄 `texture-realism-rules.md`      # Garde-fous de rendu 3D strictes : éviter complètement les conceptions tierces indésirables (comme le logo du monstre) et appliquer strictement les textures physiques réalistes (par exemple, correspondant à une noix de coco verte naturelle)
📄 `bambulab-h2d-pro-sync.md`      # Profils de trancheur et gestion de la file d'attente d'impression pour le prototypage matériel

## 📁 growth-and-content/ (3 compétences - Haut de l'entonnoir)
📄 `instagram-unboxing-scripts.md` # Modèles de storyboard haute énergie pour déballer le matériel lourd (comme un tout nouveau Nvidia DGX Spark ou une imprimante 3D) pour les Reels des médias sociaux
📄 `medium-blog-enhancer.md`       # Affine les mises à jour techniques denses en articles Medium très lisibles adaptés aux développeurs IA
📄 `linkedin-hook-writer.md`       # Rédige des crochets percutants et accrocheurs pour partager les jalons techniques sur LinkedIn
