# 📂 Studio de Création de Contenu IA

> L'espace de travail canonique pour un Créateur de Contenu Technique et Constructeur IA, conçu pour automatiser la syndication multi-canaux, rationaliser les vidéos de déballage et maintenir une esthétique de marque stricte sur YouTube, Instagram, Medium et LinkedIn.

📄 `content-strategy-brief.md` # Brief canonique : Définit les personas d'audience, les piliers techniques fondamentaux et les calendriers de publication omnicanaux
🧠 `active-campaign-memory.md` # Mémoire de session : Suivi dynamique du contexte pour les statuts de brouillons actuels et l'actualité technologique tendance
🤖 `CLAUDE.md`                 # Règles opérationnelles : Instructions strictes pour ne jamais halluciner les faits techniques et préserver explicitement les couleurs de base originales lors de l'amélioration ou du redimensionnement des images de référence produit (par exemple, peintures ou textures de site Web)

## 📁 video-and-reels-engine/ (4 compétences - Narration Visuelle)
📄 `unboxing-storyboards.md`   # Listes de prises de B-roll et scripts de rythme haute énergie pour les déballages de matériel lourd (comme un nouveau Bambulab H2d Pro ou un Nvidia DGX Spark) sur Instagram Reels
📄 `blender-youtube-scenes.md` # Invites et scripts Python pour générer automatiquement des animations Blender et des scènes 3D pour les didacticiels YouTube approfondis
📄 `retention-hook-writer.md`  # Cadres pour les 3 premières secondes d'une Reel afin de maximiser la rétention des spectateurs et d'éviter le défilement
📄 `auto-caption-styler.md`    # Configurations pour la transcription de l'API Whisper avec animations de texte dynamiques automatisées

## 📁 written-content-pipeline/ (3 compétences - Copie et Blogs)
📄 `medium-blog-enhancer.md`   # Prend des pensées denses et non structurées (par exemple, les mises à jour sur Claude Managed Agents) et les affine en articles Medium hautement lisibles et formatés
📄 `linkedin-growth-hooks.md`  # Génère des publications de texte LinkedIn percutantes et accrochées par défilement, ainsi que des spécifications de dimension de bannière personnalisées pour stimuler l'engagement professionnel
📄 `technical-tutorial-drafter.md` # Convertit la documentation API brute et les extraits de code en guides écrits pas à pas

## 📁 asset-and-design-ops/ (3 compétences - Marque)
📄 `image-enhancement-guardrails.md` # Règles strictes pour les flux de travail de mise à l'échelle IA afin de garantir que l'art original et les couleurs de peinture sont préservés à 100 % tout en améliorant uniquement la résolution pour les listes de produits du site Web
📄 `thumbnail-ab-tester.md`    # Cadres pour générer des vignettes YouTube hautement contrastées et cliquables avec des superpositions de texte variées
📄 `brand-voice-aligner.md`    # Vérifie que tout le texte généré garantit que le ton correspond à la voix authentique et centrée sur le développeur du créateur plutôt qu'au discours IA générique

## 📁 publishing-and-analytics/ (3 compétences - Distribution)
📄 `cross-platform-scheduler.md`# Mappage d'API pour pousser automatiquement le même message principal formaté nativement pour Twitter, LinkedIn et Instagram
📄 `engagement-tracker.md`     # Scripts pour extraire les comptes de vues et les taux de clics (CTR) afin d'identifier les formats de contenu qui génèrent réellement du trafic
📄 `github-final-sync.md`      # Déclencheurs automatisés pour fusionner proprement les blogs markdown finalisés et les scripts d'animation Python dans les dépôts finaux Github
