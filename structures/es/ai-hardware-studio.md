# 📂 Estudio de Hardware AI
> El espacio de trabajo canónico para un Fundador de Hardware o Diseñador de Productos 3D, diseñado para conectar la orquestación autónoma de IA con la fabricación física, el prototipado de productos y la creación de contenido social.

📄 `studio-architecture-brief.md` # Resumen canónico: Define el stack de hardware, los planes de monetización de SaaS multi-agente y los cadencias de lanzamiento de contenido
🧠 `active-fabrication-queue.md`  # Memoria de sesión: Rastreo de contexto dinámico para impresiones 3D nocturnas, ajuste fino de modelos locales y ajustes de CAD activos
🤖 `CLAUDE.md`                    # Reglas operativas: Instrucciones estrictas para preservar los colores de referencia originales durante la generación de texturas, garantizar acabados de materiales realistas (p. ej., cáscaras de coco verde natural) y eliminar explícitamente cualquier logo de terceros no deseado de modelos 3D

## 📁 hardware-and-compute/ (4 habilidades - El Stack Físico)
📄 `bambulab-h2d-pro-config.md`   # Orquestación de G-code, asignación de filamentos y configuraciones de red para la granja de impresoras principal
📄 `mac-mini-agent-host.md`       # Configuración para implementar agentes autónomos locales en un bastidor de servidor Mac mini dedicado
📄 `nvidia-dgx-spark-allocator.md`# Reglas de enrutamiento de cómputo para descargar tareas pesadas de ML local al DGX Spark
📄 `iot-sensor-telemetry.md`      # Webhooks para monitorear temperaturas de impresión y fallos en tiempo real

## 📁 cad-and-prototyping/ (3 habilidades - Diseño de Productos)
📄 `generative-mesh-enhancer.md`  # Scripts para suavizar geometrías 3D manteniendo la integridad estructural de artículos de estilo de vida funcionales
📄 `texture-and-color-rules.md`   # Barreras de seguridad que garantizan que las texturas generadas por IA se apliquen sin alucinar logos de monstruos o alterar el arte de referencia de base
📄 `pop-culture-prototyper.md`    # Restricciones de tamaño específicas y planos para artículos de novedad funcional (p. ej., soportes para botellas de vino Rick, soportes para latas o carcasas de electrónica personalizada)

## 📁 autonomous-operations/ (3 habilidades - La Fuerza de Trabajo Virtual)
📄 `zaltaclaw-overnight-loop.md`  # Motor de ejecución que permite a agentes de codificación autónomos ejecutar tareas nocturnas sin intervención humana
📄 `claude-managed-agents.md`     # Especificaciones de integración para implementar y monitorear agentes administrados en beta pública de Claude
📄 `agent-workforce-router.md`    # El nodo supervisor para una SaaS multi-agente, enrutando tareas entre personas virtuales de empleados

## 📁 social-content-engine/ (3 habilidades - Crecimiento Omnicanal)
📄 `instagram-unboxing-scripts.md`# Guiones gráficos, configuraciones de iluminación y listas de tomas para Reels de desempaquetado de hardware de alta energía (p. ej., revelar una nueva impresora 3D o sistema DGX)
📄 `medium-technical-drafter.md`  # Mejora borradores técnicos profundos en artículos de Medium altamente legibles enfocados en desarrollo de IA
📄 `youtube-blender-scenes.md`    # Indicaciones y scripts para generar animaciones de Blender y miniaturas de tutoriales de YouTube

## 📁 software-and-sync/ (3 habilidades - Infraestructura)
📄 `health-ui-frontend.md`        # Configuraciones de Next.js / React para el panel de control orientado al cliente
📄 `health-api-backend.md`        # Lógica de FastAPI / Node para el enrutamiento backend central y la coordinación de agentes
📄 `github-final-sync.md`         # Disparadores de CI/CD automatizados para fusionar de forma limpia bloques de código aprobados en sus repositorios finales de Github
