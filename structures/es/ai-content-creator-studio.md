# 📂 Estudio de Creador de Contenido AI

> El espacio de trabajo canónico para un Creador de Contenido Técnico y Desarrollador de IA, diseñado para automatizar la sindicación multicanal, optimizar videos de unboxing y mantener una estética de marca coherente en YouTube, Instagram, Medium y LinkedIn.

📄 `content-strategy-brief.md` # Resumen canónico: Define personas de audiencia, pilares técnicos fundamentales y cronogramas de lanzamiento omnicanal
🧠 `active-campaign-memory.md` # Memoria de sesión: Seguimiento de contexto dinámico para estados de borrador actuales y noticias tecnológicas tendencia
🤖 `CLAUDE.md`                 # Reglas operativas: Instrucciones estrictas para nunca alucinar hechos técnicos y preservar explícitamente los colores de línea de base originales al mejorar o aumentar la resolución de imágenes de referencia de productos (por ejemplo, pinturas de sitios web o texturas)

## 📁 video-and-reels-engine/ (4 habilidades - Narrativa Visual)
📄 `unboxing-storyboards.md`   # Listas de planos B-roll y scripts de ritmo de alta energía para unboxing de hardware pesado (como una nueva Bambulab H2d Pro o una Nvidia DGX Spark) en Instagram Reels
📄 `blender-youtube-scenes.md` # Prompts y scripts de Python para generar automáticamente animaciones de Blender y escenas 3D para tutoriales de YouTube profundos
📄 `retention-hook-writer.md`  # Marcos de trabajo para los primeros 3 segundos de un Reel para maximizar la retención de espectadores y prevenir desplazamientos
📄 `auto-caption-styler.md`    # Configuraciones para transcripción de API Whisper con animaciones de texto dinámicas automatizadas

## 📁 written-content-pipeline/ (3 habilidades - Copia y Blogs)
📄 `medium-blog-enhancer.md`   # Toma pensamientos densos y desestructurados (por ejemplo, actualizaciones sobre Claude Managed Agents) y los refina en artículos de Medium altamente legibles y formateados
📄 `linkedin-growth-hooks.md`  # Genera publicaciones de texto de LinkedIn concisas y que detienen el desplazamiento, y especificaciones de dimensión de banner personalizado para impulsar la participación profesional
📄 `technical-tutorial-drafter.md` # Convierte documentación de API sin procesar y fragmentos de código en guías escritas paso a paso

## 📁 asset-and-design-ops/ (3 habilidades - Marca)
📄 `image-enhancement-guardrails.md` # Reglas estrictas para flujos de trabajo de aumento de resolución de IA para garantizar que el arte y los colores de pintura originales se preserven al 100% mientras solo se mejora la resolución para listados de productos de sitios web
📄 `thumbnail-ab-tester.md`    # Marcos de trabajo para generar miniaturas de YouTube de alto contraste y clicables con superposiciones de texto variables
📄 `brand-voice-aligner.md`    # Verifica todo el texto generado para garantizar que el tono coincida con la voz auténtica enfocada en desarrolladores del creador en lugar del habla de IA genérica

## 📁 publishing-and-analytics/ (3 habilidades - Distribución)
📄 `cross-platform-scheduler.md`# Mapeo de API para empujar automáticamente el mismo mensaje central formateado de forma nativa para Twitter, LinkedIn e Instagram
📄 `engagement-tracker.md`     # Scripts para extraer conteos de visualizaciones y tasas de clics (CTR) para identificar qué formatos de contenido realmente impulsan tráfico
📄 `github-final-sync.md`      # Desencadenantes automatizados para fusionar de forma limpia blogs de markdown finalizados y scripts de animación de Python en repos finales de Github
