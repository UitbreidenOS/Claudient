# 📂 Centro de Prospección SDR

> El espacio de trabajo canónico para un Representante de Desarrollo de Ventas (SDR) o Representante de Desarrollo Empresarial (BDR), diseñado para automatizar la investigación de cuentas, la construcción de listas y secuencias de alcance hiperPersonalizadas.

📄 `icp-definition.md`         # Breve canónico: Perfil de Cliente Ideal, personas de comprador y señales negativas
🧠 `active-pipeline.md`        # Memoria de sesión: Seguimiento de contexto dinámico para prospectos actualmente en secuencia
🤖 `CLAUDE.md`                 # Reglas operativas: Instrucciones estrictas para evitar jerga corporativa y mantener correos electrónicos en frío por debajo de 75 palabras

## 📁 account-research/ (4 habilidades - Inteligencia Previa a la Llamada)
📄 `intent-signal-parser.md`   # Rastrea cambios de trabajo, rondas de financiación y menciones de noticias recientes
📄 `10k-analyzer.md`           # Extrae prioridades estratégicas de informes anuales (para objetivos empresariales)
📄 `tech-stack-detector.md`    # Identifica las herramientas actuales que utiliza el prospecto a través de datos de BuiltWith/Wappalyzer
📄 `persona-mapper.md`         # Mapea puntos de dolor específicos al título exacto del trabajo del prospecto

## 📁 outreach-sequences/ (3 habilidades - Toques Multicanal)
📄 `cold-email-generator.md`   # Redacta correos electrónicos de primer contacto personalizados utilizando datos de investigación de cuentas
📄 `linkedin-dm-drafter.md`    # Crea solicitudes de conexión casuales y sin ventas, y scripts de notas de voz
📄 `cold-call-scripts.md`      # Discursos dinámicos y ganchos de apertura basados en la industria

## 📁 objection-handling/ (3 habilidades - Tarjetas de Batalla)
📄 `competitor-kill-sheets.md` # Puntos de referencia rápida sobre cómo diferenciarse de los 3 principales rivales
📄 `budget-pushback.md`        # Contrarréplicas para "no tenemos presupuesto" o "llámame en Q4"
📄 `security-concerns.md`      # Breves comentarios que abordan preocupaciones de SOC2 y privacidad de datos

## 📁 pipeline-management/ (3 habilidades - Sincronización CRM)
📄 `salesforce-sync.md`        # Formato automatizado de notas de reunión para pegado directo en CRM
📄 `follow-up-scheduler.md`    # Activadores para correos electrónicos de seguimiento de 3 y 7 días
📄 `disqualification-log.md`   # Rastrea exactamente por qué una cuenta se cerró perdida para alineación de marketing futura

## 📁 weekly-reporting/ (2 habilidades - Rendimiento)
📄 `conversion-tracker.md`     # Rastrea tasas de apertura, tasas de respuesta y métricas de reuniones reservadas
📄 `a-b-test-evaluator.md`     # Compara el rendimiento de la Línea de Asunto A versus Línea de Asunto B
