# Certified Stacks Index

This directory indexes all Claudient Marketplace stacks that have achieved certification status. Certified stacks have met quantified quality standards and active maintenance commitments.

## Certification Tiers

- **Gold:** Highest tier — 200+ installs, 4.5+ rating, official endorsement
- **Silver:** Mid tier — 50+ installs, 4.0+ rating, 6+ months active maintenance
- **Bronze:** Entry tier — passes validation and human review

For detailed criteria, see [CERTIFICATION.md](../CERTIFICATION.md).

## How to Get Certified

1. Build and publish a stack on GitHub
2. Submit to Claudient Marketplace
3. Pass automated validation and human review
4. Request certification via [becoming-certified.md](../guides/marketplace/becoming-certified.md)
5. Core team evaluates against certification criteria
6. Upon approval, your stack earns a certification badge

---

## Maintenance SLAs

Certified stacks commit to specific response times:

- **Bronze:** Critical issues within 2 weeks
- **Silver:** All issues within 1 week; critical within 2 weeks
- **Gold:** All issues within 48 hours; critical within 5 business days

See [CERTIFICATION.md](../CERTIFICATION.md) for full SLA details.

---

## Removing Certification

Certification is removed if:
- Stack is abandoned (no response for 3+ months)
- Critical security issue is unpatched
- Code of conduct violation
- SLA commitments are not met

Authors can reapply after 6 months of corrections.
